# AD.Lindel

Оживляет NPC в ваших сборках Skyrim — установщик и менеджер ИИ-диалогов (SkyrimNet + SeverActions + IntelEngine) для Skyrim Special Edition 1.5.97 и Anniversary Edition 1.6.x. Бесплатно.

## Установка

1. Скачайте `AD.Lindel.zip` из раздела **[Releases](https://github.com/Archidexter/AD.Lindel/releases/latest)**.
2. ПКМ по архиву → **Свойства** → внизу поставьте галочку **«Разблокировать»** → **ОК** *(Windows помечает скачанные файлы — это убирает лишние предупреждения системы)*.
3. Распакуйте в любую папку и запустите **`AD.Lindel.exe`**.
4. Дальше — по инструкции внутри лаунчера. Лаунчер обновляется сам.

## Поддерживаемые сборки

Протестировано на **RFAB, RFAD, STB** и ванильном Skyrim. Механизм установки универсальный — должен подойти к любой сборке на Skyrim **Special Edition 1.5.97** или **Anniversary Edition 1.6.x**.
